# js-accumulate-lagcc-lab Accumulator

## Tests

Review the `accumulate.js` file inside the `lib` directory, and then the `accumulate-spec.js` file inside the `spec` directory. Review each of the tests, and then start implementing your solutions to make each test pass in `accumulate.js`.

## Background

For example, given the collection of numbers:

- 1, 2, 3, 4, 5

And the operation:

- square a number

Your code should be able to produce the collection of squares:

- 1, 4, 9, 16, 25

To run the specs follow these commands:
- first install the required node modules to run the tests
````npm install````
- to run in the command line run
````npm test -- spec/accumulate-spec.js````
### or, via the custom script in package.json
````npm run answer````
